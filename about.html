Registro de Cambios

v0.9.X - Funcionalidades definidas para cuestiones de desarrollo.

v1.0.X - Primera versión terminada para motivos de desarrollo preliminar.

v1.0.1 - Plataforma finalizada. Se añadieron nuevos cambios en cuanto a la funcionalidad del dashboard para mejorar su funcionamiento (V1.0). Se integró un combo en el módulo "Nuevo préstamo" para usuarios Prestamistas, facilitando la selección de carreras establecidas.

v1.1.1 (Actual) - Se incorporó funcionalidades de seguridad para contraseñas. Estas se cifrarán desde la instalación, y se integraron políticas de seguridad en el módulo "Administrador de usuarios" para usuarios Administradores, evitando que se presenten en texto plano. Se añadieron validaciones extra (internas) al momento de generar nuevos préstamos. Ahora se almacena el nombre del prestamista quien registra devoluciones para su posterior consulta en vales y detalles de cada préstamo individual. Se añadieron nuevos campos en el módulo "Nuevo préstamo" para especificar Area, Unidad y Responsable, y a su vez se añadió una nueva pestaña para Administrador llamada "Configuración de variables", que permite personalizar la procedencia de los deudores y el laboratorio donde se implementará esta plataforma, y se empleará para la emisión de vales. Se corrigieron errores menores de funcionalidad interna.
